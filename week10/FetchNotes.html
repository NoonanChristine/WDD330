The Window Object

The Browser Object Model

  • BOM for short - a collection of properties and methods that contain information about the browser and computer screen
  • creates pop-up windows
  • only makes sense in a browser environment.
  • can also refer to it using the keyword this in the global scope

Going Global

  • Global variables can be accessed in all parts of the program
  • properties of a global object
  • in a browser environment the global object is the window object

Dialogs

  • there are three functions that produced dialogs in the browsers: alert() , confirm() and prompt()
  • The window.alert() method will pause the execution of the program and display a message in a dialog box
  • The window.confirm() method will stop the execution of the program and display a confirmation dialog that shows the message provided as an argument and gives the options of OK or Cancel
  • The window.prompt() method will stop the execution of the program
  • most browsers allow users to disable any dialogs from repeatedly appearing

Finding out browser information

  • The window object has a navigator property that returns a reference to the Navigator object
  • The Navigator object contains information about the browser being used

Location, Location, Location

  • The window.location property is an object that contains information about the URL of the current page
  • This property is a read/write property, which means it can also be changed by assignment

Browser History

  • The window.history property can be used to access information about any previously visited pages in the current browser session
  • The window.history.length property shows how many pages have been visited before arriving at the current page
  • The window.history.forward() and window.history.back() methods can be used to navigate forwards and backwards by one page

Controlling Windows

  • A new window can be opened using the window.open() method
  • can also be assigned to a variable, so the window can then be referenced later in the code
  • The close() method can be used to close a window
  • Also possible to move a window using the window.moveTo() method
  • Can resize a window using the window.resizeTo() method

Screen Information

  • The window.screen object contains information about the screen the browser is displayed on
  • The availHeight and availWidth can be used to find the height and width of the screen
  • The colorDepth property can be used to find the color bit depth of the user’s monitor

The Document Object

  • Each window object contains a document object
  • This object has properties and methods that deal with the page that has been loaded into the window

document.write()

  • The write() method writes a string of text to the page
  • Use of document.write() is heavily frowned upon

Cookies

  • Cookies are small files that are saved locally on a user’s computer.
  • invented by Netscape as a way of getting round HTTP being a stateless protocol
  • can be used for personalizing a user’s browsing experience, storing user preferences, keeping track of user choices, authentication and tracking users
  • take the form of a text file that contain a list of name/value pairs separated by semicolons

Creating Cookies

  • To create a cookie assign it to JavaScript’s “cookie jar” using the document.cookie property
  • The document.cookie property acts like a special type of string

Changing Cookie Values

  • Value can be changed by reassigning it to document.cookie using the same name but a different value

Reading Cookies

  • To see the current contents of the cookie jar enter document.cookie

Cookie Expiry Dates

  • Cookies are session cookies by default
  • deleted once a browser session is finished
  • can be made persistent

The Path and Domain of Cookies

  • By default cookies can only be read by pages inside the same directory and domain as the file was set

Secure Cookies

  • Adding the string ; secure to the end of a cookie ensures it’s only transmitted over a secure HTTPS network

Deleting Cookies

  • set it to expire at a time in the past
  • If it's a session cookie it will expire when the tab or window is closed

Timing Functions

setTimeout()

  • accepts a callback to a function as its first parameter and a number of milliseconds as its second parameter

setInterval()

  • works similarly to window.setTimeout() except it will repeatedly invoke the callback function after every given number of milliseconds

Animation

  • setTimeOut() and setInterval() methods can be used to animate elements on a web page

requestAnimationFrame()

  • works similarly to window.setInterval()
  • makes the most of the browser’s built-in graphics-handling capabilities
  • does not run the animation when the tab is inactive
