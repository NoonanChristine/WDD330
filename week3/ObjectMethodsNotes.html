Exercises from the article "Object Methods: this"

Objects

  • Objects represent real world entities
  • Actions are represented by JavaScript functions
  • Function that is a property of an object is called its method
  • Called object-oriented programming

Method Shorthand

  • Use "this" to access an object
  • Value of "this" is the object before the "." in calling the method

"This" is not bound

  • Can be used in any function even if it's not a method of an object
  • Does not depend on where the method was declared
  • Function can be reused for different objects
  • More possibilities of mistakes
  • Value of "this" is defined at run-time

Arrow Functions have no "this"

  • if "this" is referenced, it's taken from the outer normal function
